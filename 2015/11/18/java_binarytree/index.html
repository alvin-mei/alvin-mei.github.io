二叉树的实现Java版本

发表于 2015-11-18   |   分类于 Java   |     |   阅读次数

在计算机科学中，二叉树是每个节点最多有两个子树的树结构。通常子树被称作“左子树”（left subtree）和“右子树”（right subtree）。二叉树常被用于实现二叉查找树和二叉堆。
二叉树每个节点至多有两个子节点：左子节点与右子节点。
二叉树的第i层最多有2^(i-1)个子节点。
深度为k的二叉树的所有节点数量不超过2^k-1个。
完全二叉树：深度为k都二叉树，除了第k层，其余各层子节点都达到上限。
满二叉树：深度为k的二叉树，总节点数量为2^k-1个。
